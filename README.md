# DDS StudyOS - Canal Publico de Updates

Este repositorio e o canal publico oficial de distribuicao do DDS StudyOS.

Aqui ficam apenas os arquivos necessarios para usuarios e testers:

- instaladores e pacotes publicos
- manifests de update (`stable` e `beta`)
- DLCs publicados
- guias publicos
- changelog resumido
- roadmap publico

## Downloads

- Releases: `https://github.com/Erikalellis/DDSStudyOS-Updates/releases`
- Ultimo setup: `https://github.com/Erikalellis/DDSStudyOS-Updates/releases/latest/download/DDSStudyOS-Setup.exe`

## Canais

- Stable: para usuarios finais
- Beta: para testers e homologacao

- `installer/update/stable/update-info.json`
- `installer/update/stable/dlc-manifest.json`
- `installer/update/beta/update-info.json`

- `CHANGELOG_PUBLIC.md`
- `ROADMAP_PUBLIC.md`

Este repositorio nao contem o codigo-fonte do produto. Ele existe apenas para distribuicao segura e atualizacoes publicas.
